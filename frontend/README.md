# Frontend - Hacker News Scraper

This is the frontend portion of the Hacker News Scraper application, built with React and Vite.

## Features
- Real-time story browsing
- User authentication (Login/Register)
- Bookmarking system
- Responsive Dark Mode UI

## Tech Stack
- React 19
- Vite
- Axios
- Lucide React
- CSS Modules / Vanilla CSS

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```
